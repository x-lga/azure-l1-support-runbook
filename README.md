# azure-l1-support-runbook

Azure Portal L1 support runbook covering the most common support tickets encountered in Azure cloud operations. Built from AZ-104 study and hands-on lab work in an Azure hybrid environment.

---

## Contents

| File | Purpose |
|------|---------|
| `01-vm-troubleshooting.md` | VM start failures, boot diagnostics, escalation |
| `02-entra-id-user-management.md` | Password reset, session revoke, MFA, sign-in logs |
| `03-nsg-and-network.md` | NSG rule creation, traffic diagnostics |
| `04-app-service-recovery.md` | App Service restart, slot swap, logs |
| `05-kql-quick-reference.md` | Log Analytics KQL queries for VM, auth, disk, NSG |
| `06-common-azure-error-codes.md` | Error code reference with L1 actions |

---

## Skills Demonstrated

- **Azure Portal:** VM lifecycle management, Entra ID administration, NSG rules, App Services
- **Entra ID:** Password reset, session revoke, MFA management, sign-in log analysis
- **KQL:** Log Analytics queries for performance, auth, and network events
- **RBAC:** Understanding role scope and troubleshooting permission errors
- **AZ-104:** Real-world application of Azure Administrator skills
